# 04. パフォーマンス

「動いてはいるが、なぜか遅い」アプリの裏側にはたいてい同じパターンがあります。1リクエストで何百回も SQL を発行している、必要のないデータまで全件読み込んでいる、毎回ゼロから計算し直している、など。

## このセクションのねらい

計測 → 原因特定 → 修正 → 再計測、というパフォーマンスチューニングの基本サイクルを通じて、**「速い／遅い」を感覚ではなく根拠で語る** ための見方を身につけます。

各レクチャー共通の起動手順と前提環境は [sections/README.md](../README.md) を参照してください。
