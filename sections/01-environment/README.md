# 01. 環境構築

このハンズオン全体で扱うサンプルアプリ（社内ナレッジ共有ツール）の **起動手順を体験** し、**アプリの全体像** を理解するセクションです。後続のセクションで問題を見つけて修正するためには、まずアプリの構造を知っておく必要があります。

## レクチャー

| レクチャー | テーマ |
|---|---|
| [01-environment](./01-environment/) | 環境構築・全体像の把握 |

## このセクションのねらい

具体的に身につけたい観点は次の3点です。

- **起動の流れ**: DB のテーブル作成 → シードデータ投入 → サーバー起動、という典型的な手順
- **アプリの機能**: 記事・コメント・ユーザー・管理画面・ダッシュボードがどんな動きをするか
- **コードの地図**: ルーティング・DB アクセス・テンプレート・セッション管理がそれぞれどのファイルにあるか

各レクチャー共通の起動手順と前提環境は [sections/README.md](../README.md) を参照してください。
